# smooth-doc

Ready to use documentation theme for Gatsby.

## Install

```sh
yarn add smooth-doc gatsby react react-dom
```

## Setup Gatsby

```js
module.exports = {
  plugins: ['smooth-doc'],
}
```

## Folder structure

```
src/
  images/
    home-logo.png
    logo.png
  pages/
    index.mdx
    docs/
      getting-started.mdx
```
